> Python · Llama 3 · Supabase · Telegram

An autonomous AI agent that collects anonymous confessions in Nairobi Sheng via Telegram, then generates and auto-posts content to X (Twitter) using Llama 3. Built a Telegram-based admin dashboard for content moderation and scheduling.

---

### [Bubasneakers](https://github.com/IGNER-Coder/Bubasneakers) — Street-Luxe E-Commerce
> Next.js 14 · React Context API · Tailwind CSS · Vercel

A mobile-first sneaker e-commerce platform with a custom design system (Oswald + Inter, monochrome + Electric Blue). Features a smart cart (composite keys for size variants, localStorage persistence), dynamic product routing, distraction-free checkout, and advanced sidebar/drawer filtering.

---

### [Nexus Brief](https://github.com/IGNER-Coder/nexus-brief) — Personalized Intelligence Dashboard
> Next.js 14 · TypeScript · Prisma · Vercel Postgres · Vercel Cron

A configurable information dashboard that runs automated hourly checks against live APIs (Finnhub stocks, Hacker News, NewsAPI) and surfaces only the alerts you care about. Built as a 10-day sprint prototype for a client R&D team.

### [NyumbaniGift](https://github.com/IGNER-Coder/NyumbaniGift) — Django E-Commerce Store

A full-stack e-commerce store for premium gifts, built with Django. Covers product catalog, cart, and order management.

### [Freshia Njeri Portfolio](https://github.com/IGNER-Coder/freshia-portfolio) — Artist Portfolio CMS
